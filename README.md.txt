# Análisis Exploratorio de Accidentes Laborales con Python y SQL

## Descripción

Este proyecto corresponde al trabajo final del curso de Python con SQL. Se desarrolló en Google Collab utilizando una base de datos anonimizada de accidentes laborales.

## Objetivo

Realizar un análisis exploratorio de datos (EDA), identificar la calidad de los datos, generar estadísticas descriptivas, elaborar visualizaciones con Plotly y realizar consultas SQL mediante DuckDB.

## Herramientas utilizadas

- Python
- Google Colab
- Pandas
- Plotly
- DuckDB
