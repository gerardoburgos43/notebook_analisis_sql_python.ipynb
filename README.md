# Análisis Exploratorio de Accidentes Laborales con Python y SQL

## Descripción del proyecto

Este proyecto corresponde al trabajo final del curso de Python con SQL. El análisis fue desarrollado en Google Colab utilizando una base de datos anonimizada de accidentes laborales.

## Descripción del dataset

El dataset contiene registros de accidentes laborales ocurridos entre los años 2024 y 2026. Para proteger la confidencialidad de la información, se eliminaron los datos personales de los trabajadores, conservando únicamente las variables necesarias para el análisis.

## Objetivo del análisis

Realizar un análisis exploratorio de los datos (EDA) utilizando Python, con el fin de evaluar la calidad de la información, obtener estadísticas descriptivas, elaborar visualizaciones y realizar consultas SQL que permitan identificar patrones relevantes dentro del conjunto de datos.

## Fuente del dataset

Base de datos institucional anonimizada para fines académicos.

> *No se incluye enlace debido a que el dataset no proviene de Kaggle ni de una fuente pública.*

## Etapas desarrolladas

- Carga del dataset en Google Colab mediante pandas.
- Exploración inicial de la estructura de los datos.
- Evaluación de la calidad de los datos mediante la identificación de valores nulos, vacíos y registros duplicados.
- Obtención de estadísticas descriptivas para variables numéricas y categóricas.
- Elaboración de visualizaciones con Plotly para identificar tendencias y distribuciones.
- Desarrollo de consultas SQL utilizando DuckDB sobre el DataFrame.

## Tecnologías utilizadas

- Python
- Google Colab
- Pandas
- NumPy
- Plotly
- DuckDB

## Instrucciones de ejecución

1. Abrir el archivo `notebook_analisis_sql_python.ipynb` en Google Colab.
2. Cargar el archivo `dataset.xlsx` (o `dataset.csv`) cuando sea solicitado por el notebook.
3. Ejecutar las celdas en orden, de arriba hacia abajo, para reproducir el análisis completo.
